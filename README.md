# TXT/DOCX to PDF Converter

A Streamlit application that converts `.txt` and `.docx` files into downloadable PDF documents.

## Features

* Upload TXT and DOCX files
* Edit extracted content before conversion
* Convert original or edited content to PDF
* Proper page margins on all sides
* Automatic line wrapping
* Multi-page support
* Download generated PDF instantly

## Technologies Used

* Python
* Streamlit
* python-docx
* ReportLab

## Installation

```bash
pip install streamlit python-docx reportlab
```

## Run Project

```bash
streamlit run app.py
```
